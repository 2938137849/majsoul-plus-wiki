请注意，雀魂 Plus 的所有快捷键均在载入游戏后才会生效

| 功能       | 快捷键                                               |
| ---------- | ---------------------------------------------------- |
| 截图       | `F12` / `Cmd+P` (macOS) / `Ctrl+P` (Windows / Linux) |
| 全屏       | `F5` / `F11`                                         |
| 退出全屏   | `Esc`                                                |
| 置顶窗口   | `Cmd+T` (macOS) / `Ctrl+T` (Windows / Linux)         |
| 退出程序   | `Alt+F4`                                             |
| 开发者工具 | `Cmd+I` (macOS) / `Ctrl+I` (Windows / Linux)         |
