# 使用 OBS 采集游戏画面

雀魂 Plus 支持使用 OBS 采集游戏画面，您可以启动雀魂 Plus 并进入游戏后，遵循以下步骤进行操作：

1. 在「来源」列表下点击「➕」，选择「游戏捕获」选项
2. 选中「新建」，并点击「确定」
3. 在模式中选中「捕获特定窗口」
4. 窗口选择「Majsoul_Plus.exe」
5. 窗口匹配优先级选择「匹配标题，否则查找同样可执行程序的窗口」
6. 点击「确定」
